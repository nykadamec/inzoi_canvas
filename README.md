# Inzoi Canvas Downloader

Chrome Extension — Download Inzoi Canvas creations as ZIP without running the game.

## Install

1. Download the latest `.crx` from [Releases](../../releases)
2. Open `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Drag the `.crx` file onto the page, or click **Load unpacked** and select the extension folder

## Features

- Download any Inzoi Canvas creation as a structured ZIP
- Automatic save or manual "Save As" dialog
- Required mods list (CurseForge integration)
- Canvas folder structure preserved in ZIP

## Update

Updates are automatic. The extension checks for new versions on startup. If an update is available you'll see a notification in the extension panel.

## Build from Source

Source code is not included in the public repository. If you have access to the private source, build with:

```bash
npm install
# Extension is in inzoi-canvas-extension/
```
